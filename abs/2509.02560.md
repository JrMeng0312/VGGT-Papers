### FastVGGT: Training-Free Acceleration of Visual Geometry Transformer

We analyze VGGT, a leading feed-forward 3D vision model, identifying efficiency bottlenecks during inference with extended image sequences. We observe a token collapse issue in attention mechanisms. Building on these insights, we introduce FastVGGT, implementing token merging specifically designed for 3D architectures without requiring retraining. The method employs a novel partitioning approach for tokens that removes redundant calculations while maintaining reconstruction quality. Testing across 3D benchmarks demonstrates effectiveness, with experiments using 1000 input images showing a 4x speedup over VGGT while mitigating error accumulation in long-sequence scenarios.

我们分析了领先的前馈式三维视觉模型 VGGT，识别了其在扩展图像序列推理过程中的效率瓶颈。我们观察到注意力机制中的 token 坍塌问题。基于这些洞察，我们提出了 FastVGGT，实现了专门为三维架构设计的 token 合并方法，无需重新训练。该方法采用新颖的 token 分区策略，在保持重建质量的同时消除冗余计算。在三维基准测试中的实验表明了其有效性，使用 1000 张输入图像的实验显示比 VGGT 加速 4 倍，同时缓解了长序列场景中的误差累积。
