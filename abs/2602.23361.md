### VGG-T3: Offline Feed-Forward 3D Reconstruction at Scale

We present a scalable 3D reconstruction model that addresses a critical limitation in offline feed-forward methods: their computational and memory requirements grow quadratically with respect to the number of input images. We employ test-time training to compress scene geometry representation into a fixed-size neural network. Our method scales linearly with the number of input views and achieves an 11.6x speed-up over baselines that rely on softmax attention while processing 1,000 images in 54 seconds. The model maintains strong reconstruction accuracy and demonstrates visual localization capabilities using unseen images as scene queries.

我们提出了一个可扩展的三维重建模型，解决了离线前馈方法中的一个关键限制：其计算和内存需求随输入图像数量呈二次增长。我们采用测试时训练将场景几何表示压缩到固定大小的神经网络中。我们的方法随输入视图数量线性扩展，与依赖 softmax 注意力的基线相比实现了 11.6 倍加速，处理 1,000 张图像仅需 54 秒。该模型保持了强大的重建精度，并展示了使用未见图像作为场景查询的视觉定位能力。
