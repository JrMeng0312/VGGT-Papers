### CUT3R: Continuous 3D Perception Model with Persistent State

We present a unified framework for solving various 3D tasks using a stateful recurrent model that continuously updates its state representation with each new image observation. From image streams, it generates metric-scale pointmaps (per-pixel 3D coordinates) in an online fashion within a common coordinate system, enabling accumulation into dense scene reconstructions. CUT3R captures scene priors to predict accurate pointmaps and infer unseen regions through virtual viewpoint probing. The approach handles variable image sequences — video streams or unordered photo collections — containing static and dynamic content. We demonstrate competitive or state-of-the-art performance across various 3D/4D tasks.

我们提出了一个统一框架，使用有状态的循环模型解决各种三维任务，该模型随着每张新图像观测不断更新其状态表示。从图像流中，它在公共坐标系中以在线方式生成度量尺度的点图（逐像素三维坐标），从而积累成稠密场景重建。CUT3R 捕获场景先验以预测精确的点图，并通过虚拟视点探测推断未见区域。该方法处理可变图像序列——视频流或无序照片集合——包含静态和动态内容。我们在各种三维/四维任务中展示了有竞争力的或最先进的性能。
